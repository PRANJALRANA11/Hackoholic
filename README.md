<h2>🛠️ Installation Steps:</h2>

<p>1. Go to client directory</p>

```
cd client
```

<p>2. Install Dependancies</p>

```
npm install
```

<p>3. start project</p>

```
npm start
```

<p>4. go to server directory</p>

```
cd server
```

<p>5. install dependacies</p>

```
pip install -r requirements.txt
```

<p>6. run project</p>

```
uvicorn app.main:main:8000
```
